專案簡介

這個專案模擬真實訂房網站的核心流程，訪客可以瀏覽不同房型、 選擇入住與退房日期、即時看到總價計算結果，並完成訂房送出。 目標是練習表單邏輯、日期運算與資料驗證，同時做出符合實際使用情境的響應式介面。

功能亮點

01

房型列表

依房型分類呈現卡片式列表，每張卡片顯示房型圖片、坪數、可容納人數與每晚價格，方便訪客快速比較。

CSS Grid Responsive
02

日期選擇與總價計算

訪客選擇入住與退房日期後，系統自動計算住宿天數與總價，並即時顯示在畫面上，避免訪客手動換算。

JavaScript Date 即時運算
03

訂房送出與驗證

表單送出前檢查必填欄位與日期邏輯（退房日期需晚於入住日期），並在錯誤時提示訪客修正，確保資料正確才送出。
